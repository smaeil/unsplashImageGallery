About Photo Gallery (this Project)

The steps of implemention:

Resources: all images has been downloaded from unsplash.com website, and all pictures are free to download and use.
The icon for favicon is from flaticon.com.
The rest of the icons has been downloaded form FontAwsome.com.

Basic Styling: The styling is implemented into and "style" tag at head of the hmtl document rather than having separated css file.

Darkmode: The darkmode button uses a JavaScript funciton "changemode()" to change the inner content of another styling tag in the head of the document with the id of "modes". It also changes the srouce for the icon image of "Darkmode" in order for changing of the icon.

Mimicking single page app: In oreder to have all pages in one single html file, all the pages has been arrange into dinstinct "div" elements with distincted ids. And by hornesting the power of simple JavaScript functions the display property of targeted div switches from none to block/flex, and opposite for none targeted divs

The images objects and populators: There has been 3 catagories for picture considered, and each catagory has been defined as an JavaScript Array containing the name of each picture. The function of "catDisplay(picture catagory array, picture css class)" populates the dives with small thumbnails of the pictures

Picture Viewer: When a picture's thumbnail is clicked the function of "vOpen(picture's name)" will be called with opens the "viewer div" and load selected image into the viwer div for display. In viewer div there are option to zoom in and out, downloading the picture and applying some filter to the viewed picture.
